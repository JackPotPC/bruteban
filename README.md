# bruteban

bruteban — это утилита для обнаружения и автоматической блокировки попыток brute-force атак по логам сервисов (SSH, веб-серверы и др.).

## Возможности

- Анализ логов в реальном времени
- Обнаружение повторяющихся неудачных попыток входа
- Автоматическая блокировка IP
- Поддержка нескольких источников логов
- Гибкая настройка правил
- Лёгкая интеграция с Linux firewall (iptables / nftables)

## Установка

### Требования

- Linux
- Python 3.10+
- Доступ к firewall
